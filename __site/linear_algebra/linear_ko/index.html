System of linear equations

In Field F, System of m linear equations in n unknowns is that
problem find tuple (x1, ..., x2) above field F that satisfy every linear equations of system called "solution".

\[ \begin{cases} \ A11 x1 + A12 x2 + ... + A1n xn = y1.\\ \ A21 x1 + A22 x2 + ... + A2n xn = y2.\\ \ \cdots \\ \ Am1 x1 + Am2 x2 + ... + Amn xn = ym.\\ \end{cases} \]

If y1 = y2 = ym, The system is homogeneous.
If two different system have same solution, Two systems are equivalent.

Elemination.

아래에 주어진 연립 방정식의 해를 구하기 위해서는,

\[ \begin{cases} \ 1x + 2y = 3. \\ \ 4x + 5y = 6. \\ \end{cases}\\ \]

Subtract 4 times the first equation from the second equation.
This eliminates x from the second equation. and it leaves one equation for y :

\[ (4x + 5y) - 4(1x + 2y) = -6,\ given\ -3y = -6. \]

Immediately we know y = 2. Then x comes from the first equation 1x + 2y = 3
Back-substitution : 1x + 2(2) = 3 given x = -1.

소거법은 세가지 연산을 통해 수행할 수 있으며,
아래의 세가지 연산을 elementary row operation이라고 한다.

\[ Row-multiplying\ transformation,\ e(A)_{ij} = A_{ij},\ if i \neq r,\ e(A)_{rj} = cA_{rj}.\\ Row-switching\ transformation,\ e(A)_{ij},\ if i \neq r and s, e(A)_{rj} = A_{sj}, e(A)_{sj} = A_{rj}.\\ Row-addition\ transformation,\ e(A)_{ij} = A_{ij} + c \cdot A_{sj}.\]
