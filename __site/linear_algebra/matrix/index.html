Matrix notation

미지수는 별로 중요하지 않음으로 다음과 같이 쓸 수 있다 :

\[ A = \begin{bmatrix}\ 1 & 2\ \\\ 4 & 5\ \end{bmatrix},\ X = \begin{bmatrix} x \\\ y \end{bmatrix},\ Y = \begin{bmatrix}\ 3\ \\\ 6\ \end{bmatrix} \] \[ AX = Y. \]

a_Ij : components row, column m x n matrix ( m by n matrix)

m = n => square matrix. all component = 0 => zero matrix.

We form the argumented matrix \(A'\) of the system \(AX = Y\).
m x (n+1) matrix, A'ij = Aij, if j <= n, A'i(n+1) = yi.

Scientific system described in \(AX = Y\) form, that is why vector denoted in column form. When the vector denoted in column form, it is easy to substitute to \(X\).
